# PVCertViewer

#### Run development
```bash
npm run development
```
Visit [http://localhost:3000](http://localhost:3000)

#### Run production
```bash
npm run production
```
Visit [http://localhost:3000](http://localhost:3000)

### Build and deploy to gh-pages
```bash
npm run deploy:gh-pages
```

You can see example on [page](https://peculiarventures.github.io/PVCertViewer/)
